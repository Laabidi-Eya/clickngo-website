# \# clickngo-website

# 

# \*\*Description\*\*  

# ClickNgo is a Tunisian leisure web platform that lists activities by category. Users can book online, explore events, shop related products, use carpooling, and discover sponsors. AI-powered modules personalize recommendations, making it a one-stop solution for leisure in Tunisia.

# 

# ---

# 

# \## Features

# \- Online booking for activities

# \- Event exploration

# \- E-commerce for related products

# \- Carpooling integration

# \- AI-powered recommendations

# \- Sponsors \& partners listing

# 

# ---

# 

# \## Tech Stack

# \- Frontend: HTML, CSS, JavaScript

# \- Backend: PHP, MySQL

# \- AI Modules: Python scripts / APIs

# \- Server: XAMPP (Apache + MySQL)

# 

# ---

# 

# \## How to run locally with XAMPP

# 1\. \*\*Install XAMPP\*\* from \[https://www.apachefriends.org](https://www.apachefriends.org)  

# 2\. Copy the project folder to `C:\\xampp\\htdocs\\clickngo-website`  

# 3\. Open \*\*XAMPP Control Panel\*\* and start \*\*Apache\*\* and \*\*MySQL\*\*  

# 4\. Create a database in \*\*phpMyAdmin\*\* (e.g., `clickngo\_db`)  

# 5\. Import the SQL file if you have one (tables + sample data)  

# 6\. Open your browser and go to:  

# http://localhost/Projet%20Web/mvcUtilisateur/View/FrontOffice/index.php

# 

# Copier le code

# 7\. Enjoy the app!
